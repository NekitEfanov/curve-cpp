# curve-c++

Simple implementation of a library contains a family of second-order curves.

## Table of contents

0. [Build](#Build)
1. [Examples](#Examples)

## Build

[*CMake](https://cmake.org/) version 3.1.0 or higher is required.
1. mkdir ./build
2. cd ./build
3. cmake ../ -A x64 -DCMAKE_BUILD_TYPE=[Release/Debug]
4. cmake --build ./
5. cmake --install ./
